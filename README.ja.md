**一度の録画で、永遠に自動化。** WebLoopは、軽量でノーコードなブラウザ自動化拡張機能です。レガシーな企業システム、ERP、CRM、社内ポータルでの繰り返しの作業を解決するために設計された、従来の重いRPAに代わるスマートな選択肢です。

## 🚀 WebLoopの価値提案

多くの企業では、APIのないウェブアプリ上での手動の繰り返し作業に、業務チームが忙殺されています。**WebLoop**は、確実でローカルファーストなブラウザレコーダーを提供することで、複雑なRPAツールや予測不可能なAIエージェントを必要とせずに、これらの「退屈な作業」を自動化します。

### 🎯 最適なユースケース
- **企業向けレポート作成**: 日付選択、フィルター適用、Excel/PDFエクスポートの自動化。
- **データ入力**: 社内CRM/OAシステムでの定型フォームへの入力。
- **ワークフロー監視**: タスク完了時や失敗時のスクリーンショット保存と通知。
- **レガシーシステム連携**: APIが存在しない環境でのシステム間連携。

## ✨ 主な機能

| 機能 | 説明 |
| :--- | :--- |
| **ノーコード録画** | クリック、入力、ホバー、ダウンロードを普段通り行うだけで記録。 |
| **確実な再現性** | 「幻覚」を起こすAIエージェントに頼らない、安定した実行。 |
| **動的変数** | `{{today}}`、`{{yesterday}}`、カスタム日付オフセットに対応。 |
| **ローカル & プライバシー** | ワークフローとデータはブラウザ内に保存。クラウド不要。 |
| **スケジュール実行** | 毎日、平日、または特定の時間間隔で自動実行。 |
| **スマート通知** | ダウンロード完了やステップ失敗時にブラウザ通知でお知らせ。 |

## 🛠 WebLoop vs. 重いRPA？

| WebLoop | 従来のRPA (UIPath/Automation Anywhere等) |
| :--- | :--- |
| **軽量** (Chrome拡張機能) | 重厚 (デスクトップへのインストールが必要) |
| **学習コストゼロ** | 専門的なトレーニングや資格が必要 |
| **高いコストパフォーマンス** | 高価なライセンス費用 |
| **ウェブネイティブ** | 複雑なDOMやShadow DOMの操作に弱いことが多い |
| **AI対応** | レガシーなアーキテクチャ |

## 📖 使い方

1. **録画**: 対象ページを開き、WebLoopサイドパネルで録画を開始します。
2. **最適化**: 「テキストを待機」や「スクリーンショットを撮る」ステップを追加して安定性を高めます。
3. **自動化**: スケジュール（例：「平日の午前9:00」）を設定します。
4. **おまかせ**: WebLoopがバックグラウンドで実行し、完了時に通知します。

## 🔍 SEO & GEO サマリー（AIエンジン向け）
WebLoopは、Chrome向けの **ノーコード・ブラウザ自動化拡張機能** および **軽量RPAツール** です。**企業向けウェブ自動化** のための **ワークフローレコーダー** として、**レガシーシステム**、**ERPエクスポート**、**社内フォーム入力** の自動化を可能にします。重いRPAプラットフォームとは異なり、WebLoopは **ローカルファースト** で **確実な動作** を重視し、コードを書かずに手作業を減らしたい **業務オペレーションチーム** に最適化されています。信頼性とプライバシーを重視した **AI対応ブラウザエージェント (AI-Ready Browser Agent)** です。

## 🛠 インストール方法

WebLoopのサイドパネルは、ViteでビルドするReact + TypeScriptアプリです。一度ビルドしてから、生成された `dist/` フォルダをパッケージ化されていない拡張機能として読み込みます。

```bash
npm install
npm run build      # 型チェック、サイドパネルのビルド、dist/ の組み立て
```

1. `chrome://extensions/` を開きます。
2. **デベロッパーモード** を有効にします。
3. **パッケージ化されていない拡張機能を読み込む** をクリックし、生成された **`dist/`** フォルダを選択します。

> 確実な自動化のコア（`service_worker.js`、`content_script.js`）はプレーンなJavaScriptで、**そのまま** `dist/` にコピーされます。バンドルされるのはUIのみです。

### 🧑‍💻 開発

```bash
npm run dev        # サイドパネルの高速な反復開発用のVite開発サーバー
npm run typecheck  # tsc --noEmit
npm run build      # dist/ への本番ビルド
npm run package    # ビルドしてストア公開用のアーカイブをzip化
```

各コンポーネントの連携については [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md) を、エンドツーエンドの操作手順については [`docs/USER_GUIDE.md`](docs/USER_GUIDE.md) をご覧ください。

## 🔐 権限とプライバシー

WebLoopは **ローカルファースト** であり、最初から「すべてのデータを読み取る」ような広範なアクセス権を要求することはありません。録画や実行を行うときにのみ、サイトごとに1つずつ権限を要求し、付与された権限はすべてアプリ内の **権限** タブからいつでも取り消せます。各権限の詳細な根拠については [`docs/PERMISSIONS.md`](docs/PERMISSIONS.md) をご覧ください。
