**一次录制，永久自动。** WebLoop 是一款轻量级、无代码的浏览器自动化插件。它专为解决企业内部旧版系统、ERP、CRM 及内网平台的重复性工作而设计，是传统重型 RPA 的精简替代方案。

## 🚀 WebLoop 的核心价值

在许多企业中，业务团队仍需在缺乏 API 的网页应用上进行大量重复的手动操作。**WebLoop** 通过提供确定性的、本地优先的浏览器录制器，帮助您处理这些“枯燥的工作”，而无需复杂的 RPA 工具或不可预测的纯 AI 代理。

### 🎯 理想使用场景
- **企业报表自动化**：自动选择日期、应用筛选条件并导出 Excel/PDF。
- **数据录入**：在内部 CRM/OA 系统中填充循环表单。
- **流程监控**：在任务完成或失败时自动截屏并通知团队。
- **旧系统集成**：在没有 API 的情况下实现系统间的流程衔接。

## ✨ 核心特性

| 特性 | 描述 |
| :--- | :--- |
| **无代码录制** | 像平常一样操作，即可捕获点击、输入、悬停和下载。 |
| **确定性回放** | 运行稳定可靠，不依赖于“会产生幻觉”的 AI 代理。 |
| **动态变量** | 支持 `{{today}}`、`{{yesterday}}` 或自定义日期偏移，适配各类报表。 |
| **本地与私密** | 所有的工作流和数据均存储在浏览器本地，无需强制上云。 |
| **定时任务** | 支持每日、工作日或特定间隔自动运行工作流。 |
| **智能通知** | 下载完成或步骤失败时，通过浏览器弹出实时提醒。 |

## 📖 工作原理

1. **录制**：打开目标页面，在 WebLoop 侧边栏启动录制。
2. **优化**：添加“等待文字出现”或“捕获截图”等步骤，增强稳定性。
3. **自动化**：设置计划任务（如“每个工作日早上 9:00”）。
4. **放松**：WebLoop 在后台运行，并在完成后通知您。

## 🛠 安装指南

WebLoop 的侧边栏是一个使用 Vite 构建的 React + TypeScript 应用。请先构建一次，然后将生成的 `dist/` 文件夹作为已解压的扩展程序加载。

```bash
npm install
npm run build      # 类型检查、构建侧边栏并组装 dist/
```

1. 打开 `chrome://extensions/`。
2. 开启 **开发者模式**。
3. 点击 **加载已解压的扩展程序** 并选择生成的 **`dist/`** 文件夹。

> 确定性自动化核心（`service_worker.js`、`content_script.js`）为纯 JavaScript，会被**原样**复制到 `dist/` 中——只有 UI 部分会被打包。

### 🧑‍💻 开发

```bash
npm run dev        # 启动 Vite 开发服务器，便于快速迭代侧边栏
npm run typecheck  # tsc --noEmit
npm run build      # 生产环境构建，输出到 dist/
npm run package    # 构建并打包为可上架商店的压缩包
```

请参阅 [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md) 了解各部分如何协同工作，并参阅 [`docs/USER_GUIDE.md`](docs/USER_GUIDE.md) 获取端到端的完整操作指南。

## 🔐 权限与隐私

WebLoop 坚持 **本地优先**，绝不会在一开始就索取宽泛的“读取你的所有数据”权限。它只在你录制或运行某个站点时，按需逐个站点请求权限，且每一项授权都可在应用内的 **权限** 标签页中随时撤销。每项权限的逐条说明请参阅 [`docs/PERMISSIONS.md`](docs/PERMISSIONS.md)。

## 🗺 路线图与演进 (Roadmap)

WebLoop 正在从一个简单的录制器，演进为面向旧系统的韧性、AI 辅助自动化伙伴。

### 🟢 阶段 1：基础建设 (当前版本)
- **零代码录制**：高保真捕获复杂的网页 UI 交互。
- **动态变量**：内置支持 `{{today}}`、`{{last_month}}` 及自定义日期偏移。
- **本地优先隐私**：完全本地执行，确保企业数据安全。
- **通用 LLM 配置**：预置支持 OpenAI, Anthropic, DeepSeek 和 Gemini。

### 🟡 阶段 2：智能与可靠性 (2026 Q3)
- **AI 自我修复**：利用 LLM 语义分析，自动检测并修复失效的选择器。
- **可视化逻辑编辑器**：直观的拖拽界面，支持条件分支 (If/Else) 和循环。
- **批量数据处理**：支持将 Excel/CSV 数据导入并自动填充网页表单。
- **静默执行**：在后台运行计划任务，不干扰用户的当前操作。

### 🔵 阶段 3：规模化与协作 (2027)
- **安全团队同步**：为部门和团队提供加密的工作流共享功能。
- **人工介入 (HITL)**：在遇到 2FA/MFA 或验证码时自动暂停并通知用户，处理后继续。
- **深度失败分析**：语义化日志分析，精准解释旧系统页面上的执行失败原因。
- **跨应用编排**：通过外部 Webhook 或本地脚本触发 WebLoop 流程。
